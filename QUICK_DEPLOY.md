# ⚡ Rychlý Netlify Deployment (bez stahování)

## Krok za krokem (5 minut):

### 1️⃣ Otevři Netlify
Jdi na: https://app.netlify.com/

### 2️⃣ Přihlaš se
- Email: tereza.kucko@gmail.com
- Heslo: [tvoje heslo]

### 3️⃣ Nový site
Klikni: **"Add new site"** → **"Import an existing project"**

### 4️⃣ Vyber GitHub
Klikni: **"Deploy with GitHub"**

(Pokud poprvé, autorizuj Netlify přístup k GitHubu)

### 5️⃣ Vyber repozitář
Najdi a vyber: **`terezakucko-eng/terka`**

### 6️⃣ Nastavení
```
Branch to deploy: claude/competitor-order-tracker-1H0ek
Build command: (nechej prázdné)
Publish directory: .
```

### 7️⃣ Deploy!
Klikni: **"Deploy site"**

Počkej 1-2 minuty a máš hotovo! 🎉

```
https://magical-unicorn-abc123.netlify.app
```

1. Site settings → Site details
2. Change site name
3. Zadej: `konkurence-ruzovy-slon`
4. Nová URL: `https://konkurence-ruzovy-slon.netlify.app`

## 📱 Po deploymentu

Otestuj aplikaci:
1. Otevři URL v prohlížeči
2. Klikni "⚙️ Správa dat"
3. Vlož URL tvé Google Sheets tabulky
4. Klikni "Importovat z Google Sheets"
5. Zkontroluj, že se data načetla správně
