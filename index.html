💡 Co je Firestore?

Cloudová databáze pro sdílení dat mezi prohlížeči a zařízeními.

✅ Připojeno: Automatická synchronizace

⚠️ Nepřipojeno: Data jen v tomto prohlížeči

📋 Jak nastavit Firestore pravidla

1. Otevři Firebase Console

2. Vyber projekt: rs-sledovani-konkurence

3. V levém menu klikni na Firestore Database

4. Přejdi na záložku Rules

5. Nahraď pravidla těmito:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

⚠️ Umožňuje přístup všem! Pro produkci použij autentizaci.

6. Klikni na Publish

🗑️ Vymazat všechna data

Vymaže všechna data z prohlížeče i Firestore. Tato akce je nevratná!
